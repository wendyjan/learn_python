Lists store items in order, and you can look up any item in the list according to its position, or "index". Lists can be sliced, just like strings.

Now let's take a look at Python's built-in method "range". This is one of my favorite list shortcuts. In Python 2.7, using the "range" method will return a list of all the integers between 0 and 1 less than the parameter. In this small example, "range" gives us access to [0, 1, 2, 3, 4]. There are 4 elements total, from 0 to 4.

Lists are mutable, which means that we often update them by adding and taking off items.

Commands for updating lists

      append - adds item to the end of a list
      del - deletes an item based on it's index
      remove - deletes an item named by item
      pop - deletes the last item or an item by index, can be printed or assigned
      insert - inserts an item at index that you choose
We also can replace an item in a list by simply using assignment(=) along with the index. Try these examples in the shell.
