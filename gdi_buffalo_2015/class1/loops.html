Control Flow with while

  • break will break out of the loop and go to the next statement outside of the loop without executing anything else within the loop.
  • continue will return to the top of the loop.
  • else will execute if the loop condition fails. Gets skipped if there is a break.

Warning: Watch out for infinite loops

DO NOT TYPE THIS x = 11 while x > 10: print “uh oh” DO NOT TYPE THIS while True: print “uh oh” To prevent infinite loops, there should be a condition in the while statement that will eventually be met OR there should be a break, usually in conjunction with if/then
